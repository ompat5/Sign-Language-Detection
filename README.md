# Sign-Language-Detection

Built an application that allows the client to make sign-language gestures in front of a webcam and then prints the corresponding letter.

## Technologies Used
- `Python 3.12`
- `TensorFlow 2.17.0`
- `Numpy 1.26.4`
- `OpenCV 4.10.0`
- `Keras 3.5.0`
